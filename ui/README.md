# Home Library UI

## Introduction
This is a user interface for a home library management system. It allows users to add, edit, and delete books from their library.

## Features
- Add new books to the library
- Edit existing books in the library
- Delete books from the library
- Search for books in the library
- Sort books by title, author, or publication date
- View book details

## How to Use

### Installation

`npm install`

### Usage

`npm start`

### Development

`npm run dev`
